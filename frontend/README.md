# Frontend

Next.js App Router dashboard for logging in with Firebase email links, linking Telegram, and reviewing assignment solutions.

## Local Run

```bash
npm install
npm run dev
```

## Vercel Deployment

1. Import the `frontend/` folder into Vercel.
2. Add all `NEXT_PUBLIC_*` environment variables.
3. Set the production domain in Firebase Authentication authorized domains.
4. Redeploy and test the email link sign-in flow.
